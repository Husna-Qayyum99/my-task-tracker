# 📝 Daily Task Tracker

A simple and clean task tracking app made with HTML, CSS, and JavaScript.

## 🚀 Features
- Add tasks
- Mark tasks as completed (strike-through)
- Delete tasks
- Task counters: total, completed, pending

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (DOM, Events, Functions)
